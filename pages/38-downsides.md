# Downsides — be honest

## More freedom means more effort

<div class="my-8">

- 🛠️ **More freedom = more effort** — you write every line yourself
- 🐌 **Performance** — no push-down to HANA; you fetch and assemble in ABAP
- 🧪 **Testing** — provider classes need real unit tests
- 🔍 **Discoverability** — colleagues won't *see* a table to start from
- 📊 **No free analytics** — KPI / aggregations need extra work

</div>

> Use them when you **need** them — not because they're cool.

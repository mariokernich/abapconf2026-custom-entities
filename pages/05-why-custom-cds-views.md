# Why Custom CDS Views?

## When a regular view simply cannot do the job {.mb-8}

- 🚫 **No data source** — data is computed, not stored
- 🔀 **Mixed entities** — combine sources that don't join nicely
- 🧮 **Calculated fields** — without 200-line `CASE WHEN` cascades
- 🌐 **API calls** — fetch from an external service at runtime
- 🎨 **Freestyle entity contents** — full control of the response

# When to pick what?

<h2 class="mb-8">A quick decision table</h2>

| Situation                                   | Use                   |
|---------------------------------------------|-----------------------|
| Data lives in a table, simple shape         | **Standard CDS View** |
| Mostly table data, a few calculated fields  | **Standard CDS View** |
| Data from an external API                   | **Custom CDS View**   |
| Computed / virtual entities                 | **Custom CDS View**   |
| Mash-up across unrelated sources            | **Custom CDS View**   |
